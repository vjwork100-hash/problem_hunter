# Privacy Policy - Problem Hunter

## Data Collection

Problem Hunter collects only publicly available data from Reddit:
- Post titles
- Post body text
- Upvote counts
- Comment counts
- Timestamps
- Subreddit names

## What We DON'T Collect

- User personal information
- Private messages
- Restricted/private subreddit content
- User browsing history
- User IP addresses

## Data Usage

Data is used exclusively for:
- Educational research into community discussions
- Pattern analysis for understanding common workflow challenges
- Learning about natural language processing techniques

## Data Storage

- All data is stored **locally** on the user's machine
- Cache files contain only post metadata for deduplication
- No cloud storage or external databases
- Data is not shared, sold, or redistributed

## Data Retention

- Cache files can be deleted at any time by the user
- No long-term data retention
- Data is used only during active research sessions

## Third-Party Services

- **Reddit API**: Used to fetch public posts (subject to Reddit's Privacy Policy)
- **Google Gemini API**: Used to analyze text patterns (subject to Google's Privacy Policy)

Neither service receives personal user data from this tool.

## User Rights

Since no personal data is collected, there is no user data to access, modify, or delete. All data analyzed is already publicly available on Reddit.

## Compliance

This tool complies with:
- Reddit's API Terms of Service
- Reddit's Responsible Builder Policy
- General data protection best practices

## Contact

For questions about data handling, please open an issue on GitHub.
