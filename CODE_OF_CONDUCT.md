# Code of Conduct - Ethical Use Guidelines

## Guiding Principles

Problem Hunter is designed as an educational research tool. By using this software, you agree to adhere to the following ethical guidelines.

### 1. Respect for Community Members
- **No Harassment**: Do not use data collected by this tool to target, harass, or spam individual users.
- **Privacy First**: Respect the privacy of Reddit users. Do not attempt to de-anonymize users or correlate data with other private sources.
- **No Doxxing**: Never publish or share personally identifiable information (PII) discovered during research.

### 2. Responsible Data Use
- **Read-Only**: This tool is for observation and analysis only. Do not use automated scripts to post, comment, upvote, or DM users based on this data.
- **Aggregation**: Focus on aggregate trends and patterns, not individual user histories.
- **No Redistribution**: Do not sell or publicly redistribute raw datasets harvested from communities without explicit permission.

### 3. Platform Compliance
- **Terms of Service**: You must comply with Reddit's User Agreement and API Terms of Service.
- **Rate Limits**: Respect API rate limits. Do not attempt to bypass throttling or authentication mechanisms.
- **Community Rules**: Respect the specific rules of each subreddit you analyze.

### 4. Educational Intent
- This tool is intended for learning about market research and NLP.
- It should not be used for commercial surveillance or competitive intelligence gathering that violates community expectations.

## Violations

Users found using this tool for malicious purposes, including spamming, harassment, or data misuse, are acting against the spirit of this project and are solely responsible for their actions.

## Reporting

If you witness unethical use of this tool or believe it violates these guidelines, please report it via GitHub Issues for review.
